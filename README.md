# MyKeymap 个人配置方案

### 💖相对来讲有用的函数
- `createNullFile` - 创建空文件或文件夹, 及时刷新视图**(不闪烁)**后, 选中并进入重命名模式**(无延迟)**
- `GetExplorerPath` - 获取当前目录或选中项路径, 支持多标签页面
- `GetExplorerPathX` - 获取当前目录或选中项路径以及详细状态, 支持多标签页面
- `openWithPwsh` - 任何场景打开基于 **Terminal** 的 **Powershell**，与 `GetExplorerPath` 深度耦合，已降权为非管理员模式，低延迟极速启动
